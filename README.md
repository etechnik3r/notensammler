# 🎵 Notensammler

Ein kleines Retro-Musikspiel für Kinder im Pixel-Art-Stil — alles in einer einzigen HTML-Datei, ohne Installation.

## ▶️ Direkt spielen

**[Jetzt im Browser spielen](http://etechnik3r.github.io/notensammler/notensammler.html)**

Läuft auf PC (Pfeiltasten ← →) und Handy/Tablet (mit dem Finger wischen).

## 🎮 Worum geht's

Fang mit dem Kescher die Noten, die vom Himmel fallen, und bring sie zum Mischpult — aber weich den Blitzen aus, sonst ist der Kescher leer.

- **Melodie-Sammler:** Ein Kinderlied vorgegeben, Noten in der richtigen Reihenfolge sammeln. Komplett = goldener Notenschlüssel. 🏆
- **Eigene Melodie:** Frei komponieren aus Achtel-, Viertel-, halben und ganzen Noten sowie **Pausen**. Am Ende bewertet das Mischpult dein Lied und vergibt einen von 24 Pixel-Pokalen.

## ✨ Aktuelle Version

- ⏸️ Sammelbare Pausen im Eigen-Modus; ganze Noten kommen seltener (wie in echter Musik)
- 🎨 Mehrere Bühnen-Hintergründe, die pro Spiel zufällig wechseln
- 🌟 Animierter Startbildschirm: schwebende Noten, tanzendes Männchen, dynamisch wogender Titel
- 🔤 Durchgehend Pixelschrift ("Press Start 2P" + "Pixelify Sans")
- 📱 Für Desktop und Mobil optimiert
- 🔊 Sound komplett über die Web Audio API (Chiptune, keine Audiodateien)

## 🛠️ Technik

Eine HTML-Datei, Vanilla JavaScript, Canvas 2D — kein Build, keine Dependencies. Alle Sprites sind als Pixelraster im Quellcode definiert.
